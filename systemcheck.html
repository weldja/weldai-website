System Check

For prospective customers · Takes about 30 seconds · No installation required

Before your Weld AI installation call, please run this quick system check. It checks your Windows version, RAM, Docker, and internet connectivity — then saves a text file to your Desktop for you to email back to us.

⬇ Download System Check Script

A PowerShell script — no installation required

Download weldai-diagnostics.ps1

How to run it

1

Download the script above

Save weldai-diagnostics.ps1 to your Desktop.

2

Right-click → Run as Administrator

Important: you must right-click the file and choose Run as Administrator — not just "Run with PowerShell". Administrator rights are needed for several of the checks.

If Windows asks about execution policy, type R and press Enter.

4

Email the results file

A file called weldai-diagnostics.zip will appear on your Desktop. Email it to hello@weldai.uk with the subject: Diagnostics - [your company name].

What does the script check? Windows version · RAM · CPU · Docker · Internet connectivity to Claude AI · Network drives and IP addresses · Available ports. It does not collect any personal data, passwords, or document contents.

We will review your results within one working day and confirm whether your system is ready for installation.
