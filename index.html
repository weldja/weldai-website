Common questions

Frequently asked questions

Everything you need to know before booking a demo.

Do my documents leave my server? ＋

No. Your documents remain on your server at all times. No full document uploads — ever. Only the question you ask and a few relevant paragraphs are sent to the AI to generate an answer. Everything else stays on your infrastructure.

What servers does it run on? ＋

Weld AI supports four install configurations:

  • Windows PC or Workstation — Windows 10 (21H2+) or Windows 11, 8GB RAM minimum. Most common for small firms.
  • Windows Server + Ubuntu VM — recommended for servers. A small Linux virtual machine runs inside your Windows Server via Hyper-V. Full network share support.
  • Linux Server — Ubuntu 22.04 or Debian 12. Simplest install, full network share support.
  • Windows Server Direct — alternative only, requires complex WSL2 setup. Ubuntu VM approach preferred.
→ Full requirements, install types and IT checklist on the System Check page
→ IT team? Step-by-step setup instructions in the IT Setup Guide
What's the install process like, and do we need IT staff? ＋

Usually under one hour, remote over a Teams or Zoom screen-share. No engineer visit, no separate IT project. In most cases your team can start asking questions the same day. Day-to-day use needs no technical skill — staff open a browser and type a question, and adding new documents is just copying a file into a folder and clicking Refresh.

What document formats does it support? ＋

PDF, Word (.docx, .doc), PowerPoint (.pptx), Excel (.xlsx, .csv), and plain text files. Most business documents are supported out of the box.

What happens when I add new documents? ＋

Copy the file into your documents folder and click Refresh in the app. Only new or changed files are re-indexed — the process takes seconds for most documents.

How much does it cost to run? ＋

Installation is from £350 (remote, includes 30 days support). The only ongoing cost is the AI usage — typically £2–5 per month for a small business, paid directly to the AI provider. No monthly subscription to Weld AI.

Where can our documents live? OneDrive, network drives, file servers? ＋

Local drives (C:, D:, E:, USB) work as long as they're connected at startup.

OneDrive or SharePoint synced to the PC works, provided the documents folder is set to "Always keep on this device" in File Explorer. Cloud-icon files aren't stored locally and can't be indexed until downloaded.

Network mapped drives aren't supported on Windows installs due to a Docker on Windows limitation. If your documents only live on a file server, we install Weld AI directly on the server (Windows Server or Linux) — that gives full network-share support.

We'll advise on the best approach during the pre-installation review.

Is this suitable for our industry? ＋

Weld AI works best for businesses with a significant volume of internal documents — policies, procedures, contracts, technical manuals, compliance guides. It works particularly well for accountancy practices, solicitors, HR firms, and engineering companies. If your team regularly spends time searching for information they know exists, it's a strong fit.
