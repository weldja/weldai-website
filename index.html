AI options

Choose the AI engine that fits your business

Weld AI works with two AI backends. We recommend Claude API for most businesses — it delivers faster, higher-quality answers. For organisations with strict data sovereignty requirements, a fully local option is available.

⭐ RECOMMENDED

Claude API

Powered by Anthropic's Claude — the leading AI model for business use

  • Fastest response times (1–3 seconds)
  • Highest accuracy and reasoning quality
  • Your documents stay on your server — only your question and relevant text excerpts are sent
  • Anthropic does not train on API data — your content is never used to train AI models
  • Anthropic is SOC 2 Type II certified
  • Requires internet connection on the server
  • Small per-query API cost (typically pennies per day)

Fully Local (Ollama)

100% on-premises — no data leaves your building, ever

  • Completely air-gapped — works with no internet
  • Zero data leaves your server
  • No API costs — unlimited queries
  • GDPR compliant with no DPA needed
  • Slower responses (5–15 seconds)
  • Lower answer quality than Claude
  • Requires a more powerful server (8GB+ RAM)

ℹ️ With Claude API: your documents never leave your server. Only the question you ask and the most relevant document extracts (a few paragraphs) are sent to Anthropic to generate the answer. Anthropic's API terms explicitly prohibit using API inputs to train their models.

Data privacy

Designed with data privacy at its core

Whether you choose Claude API or the fully local option, Weld AI is built to minimise data exposure and keep your business information under your control.

🗄️

Documents stay on your server — always

Your files are never uploaded anywhere. Only short, relevant text excerpts are sent to Claude to generate an answer — and only when using the Claude API option.

🚫

Anthropic does not train on your data

Anthropic's API Terms of Service explicitly prohibit using API inputs and outputs to train AI models. Your queries are processed and discarded — not retained or learned from.

🔒

Encrypted in transit

HTTPS encryption between browsers and your server, and between your server and the Claude API. Secure on your internal network and externally.

🏠

Fully local option for maximum privacy

For organisations that require zero external connections — solicitors handling highly sensitive matters, healthcare, classified environments — the fully local option means absolutely nothing leaves the building.

🛡️

Anthropic is SOC 2 Type II certified

Anthropic maintains enterprise-grade security certifications. Their infrastructure is built to the same standards as other enterprise cloud providers your business already trusts.

📋

Optional audit logging

Usage logging available for compliance and internal audit — so your IT team can monitor who asked what, and when.
