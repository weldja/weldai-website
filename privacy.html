2. How the Weld AI software handles your data

The Weld AI software is installed and runs entirely on your own server. The following table shows exactly what stays on your server and what is sent externally:

DataStays on your serverSent externally
Your documents (PDF, Word, Excel etc.)✓ Always✗ Never
Document index (vector database)✓ Always✗ Never
Conversation history (stored)✓ Always✗ Never sent to Weld AI

Weld AI does not have access to your documents, queries, conversation history, or API key at any time.

3. Anthropic Claude API

When using the Claude API integration, your questions and relevant document excerpts are sent to Anthropic's API to generate answers. Anthropic makes the following commitments:

  • No training on API data — Anthropic's Terms of Service explicitly prohibit using API inputs and outputs to train their AI models. Your queries are processed and discarded.
  • SOC 2 Type II certified — enterprise-grade security certification.
  • HTTPS encrypted — all communication is TLS encrypted in transit.
  • GDPR compliant — Anthropic operates under a Data Processing Agreement with Standard Contractual Clauses for EU/UK data.

For full details, see Anthropic's Privacy Policy and API Terms of Service.

For solicitors and regulated firms: As the data controller, your firm remains responsible for ensuring that sending document excerpts to Anthropic's API is consistent with your client confidentiality obligations. We recommend reviewing Anthropic's Data Processing Agreement before processing client-specific documents.
